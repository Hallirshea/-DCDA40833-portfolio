Dark Mode Feature

The dark mode works by using JavaScript to toggle a class called dark-mode on the body element. When that class is added, the CSS styles change the background and text colors. The classList.toggle() method turns the class on and off when the button is clicked. LocalStorage is used so the theme preference stays saved after the page refreshes.
