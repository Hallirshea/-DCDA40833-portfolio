Lab 5: Vibe Coding with AI

Lab Requirements

In Lab 5, the goal was to experiment with AI-assisted coding while still making sure I could understand, explain, and modify the code myself. The lab focused on using AI as a creative and technical support tool, while also thinking about responsibility, authorship, and how much understanding I actually had of the code I was using.

What I Learned

This lab showed me that AI can be really helpful when coding feels overwhelming, especially for getting started or working through something I do not fully understand yet. At the same time, it made me realize that I still need to go through the code carefully, make changes myself, and actually understand what is happening. The biggest thing I learned was that using AI well still requires effort, judgment, and active learning on my part.

What This Demonstrates

This lab demonstrates my ability to use AI as a tool without depending on it blindly. It shows adaptability, problem-solving, and a growing ability to work with code in a more thoughtful and hands-on way.

Technical Understanding

Dark Mode Feature

The dark mode works by using JavaScript to toggle a class called dark-mode on the body element. When that class is added, the CSS styles change the background and text colors. The classList.toggle() method turns the class on and off when the button is clicked. LocalStorage is used so the theme preference stays saved after the page refreshes.

Python Visualization

The visualization uses matplotlib to create a bar chart from a small dataset defined directly in the notebook. I understood how the lists store the age groups and their values, and how plt.bar() generates the chart. I modified the AI-generated output by changing the title and adjusting the data values.
